# Music Player

## Description
A simple music player built with HTML, CSS, and JavaScript. It allows users to play, pause, skip, and control music progress bar.

## Features
- Play and pause music.
- Previous and next song controls.
- Displays current song title.
- Rotating album image animation.

## Technologies Used
- HTML  
- CSS  
- JavaScript
